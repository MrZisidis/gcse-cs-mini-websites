Features
Core Concepts: Teaches the difference between the "global" turtle and creating a turtle "object" (instance).
Guided Challenge: A step-by-step task (Parts A and B) to build a spiderweb, reinforcing the use of functions and for loops.
Embedded Code: Includes skeleton code for students to start with and complete solutions for them to check against.
Interactive Quiz: A "Check Your Understanding" section with multiple-choice, fill-in-the-blank, and true/false questions.
Self-Marking: A "Mark My Work" button that instantly calculates the student's score, provides a percentage, and lists specific topics for revision.
Printable Report: A "Generate Report" button that creates a printable summary of the student's score, feedback, and submitted answers, which is ideal for assessment.
